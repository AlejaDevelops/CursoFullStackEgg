ERRORES COMUNES EN COFIFICACION HTML

  • Pretender editar o poner "bonita" la página web desde HTML. La función de este código solo es ESTRUCTURAR no EMBELLECER, para esto último se usa CCS. Lo importante de HTML es codificar con una semantica o estructura correcta, de esta forma será atrativa para Google.
  • Usar etiquetas obsoletas. Google castiga el uso de etiquetas desactualizadas. Se pueden buscar y mentenerse actualizado por internet.
  • Usar correctamente la semántica HTML
  • No mezclar lenguajes. Modularizar, separar contenido - código
