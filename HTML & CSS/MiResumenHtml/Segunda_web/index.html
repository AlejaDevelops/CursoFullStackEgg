Página web desarrollada como ejemplo práctico del HTML semántico SUBTITULO

El atributo "id=XX" se utiliza como identificado de un único objeto, el nombre debe ser único para el objeto. El identificador "Class=XX" se puede utilizar par identificar un grupo de objetos similares o de igual categoría dentro dce la página, estos sí pueden llevar el mismo nombre, ejemplo: "subtítulos". Esta selección o marcado es importante para hacer el "llamado" de estos objetos más fácilmente y permitir su posterior edición.
